Mostrar membres de cada grup

Podem escollir entre les següents opcions:

  • Abans de triar un grup: Els alumnes veuran en tot moment les persones que formen cada grup.
  • Després d'haver triat un grup: Els alumnes només veuran les persones que formen el seu grup, després d'escollir-ne un.
  • Quan es tanqui l'activitat Tria grup: Els alumnes només veuran les persones que formen el seu grup, quan l'activitat arribi a la data de tancament.
  • Mai: Els alumnes mai podran veure les persones que formen cada grup.
